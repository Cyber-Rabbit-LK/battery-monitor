# battery-monitor
Firebase Console එකට ගිහින් project එකක් හදන්න

Realtime Database එක create කරන්න (test mode එකෙන් පටන් ගන්න)

Project settings > Service accounts එකෙන් Database secrets එක copy කරගන්න

Project settings > General එකෙන් Web app එකක් register කරලා firebaseConfig එක copy කරගන්න

Electron App එක run කරන්න:
npm install electron systeminformation firebase
npm install
npm start
Web Dashboard එක බලන්න:
web-dashboard folder එකේ තියෙන index.html එක browser එකකින් open කරන්න

විශේෂාංග:
✅ Live data sync - Firebase Realtime Database එකට තත්පර 10කට වරක් data send වෙනවා

✅ Animated UI - Battery level bar, number counting animations

✅ Real-time chart - Last 10 updates එක graphical එකකින් පෙන්වයි

✅ Update log - එන සියලුම updates list එකක් ලෙස පෙන්වයි

✅ Responsive design - Mobile එකෙනුත් හොඳට පේනවා

✅ Connection status - Firebase connection status එක live එකෙන් පෙන්වයි
